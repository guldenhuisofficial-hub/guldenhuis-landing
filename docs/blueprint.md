# **App Name**: GuldenHuis Landing

## Core Features:

- Sticky Navbar: Implement a sticky navigation bar that includes the logo, menu items (Home, Produk, Portofolio, Tentang, Contact), and a CTA button linking to Instagram. Provides smooth scrolling to different sections upon menu item clicks.
- Hero Section: Create a hero section with a two-column layout. The left column will feature a headline, a brief description, and two CTAs (one to view the portfolio and another to consult via WhatsApp). The right column will display a large hero image and a stats row highlighting project accomplishments and experience.
- Services Section: Design a services section with a title, subtext, and a grid of three cards (Desain Interior, Build & Renovasi, Konsultasi & Perencanaan). Each card will include an icon, a brief description, and a link to learn more, scrolling to the portfolio/product section.
- Product/Portfolio Category: Develop a product/portfolio category section showcasing products/services in a grid of six cards. Clicking each card opens a modal or inline detail section, with a category description, mini gallery, and links to Instagram and Tokopedia.
- Featured Portfolio: Showcase a featured portfolio section presenting 6-9 portfolio cards, with optional filter chips to categorize projects (e.g., All, Kitchen, Office). Clicking a card will display a detailed modal or inline section with a slider/gallery, short description, and CTAs to Instagram and e-commerce sites.
- Testimonials: Incorporate a testimonials section with three testimonial cards featuring star ratings and short quotes. Include a button to view more reviews on Tokopedia.
- Contact Section: Implement a contact section with a two-column layout: one column for office and workshop addresses, and another with link icons for WhatsApp, Instagram, Tokopedia, Shopee, TikTok, and Facebook. A mini-menu footer will also be added.

## Style Guidelines:

- Primary color: Pale yellow (#FAF9F6) to serve as an off-white base for main sections, ensuring a clean and modern look.
- Background color: Soft Yellow (#FAF4E6), a slightly more saturated tint, used to give subtle tonal variation to different sections while keeping a light appearance.
- Accent color: Gold (#D4AF37) for highlights, icons, and accents, adding a touch of premium feel; this is approximately 30 degrees away from the primary hue.
- Headline font: 'Montserrat', a sans-serif font, for headings to maintain a modern and premium feel.
- Body font: 'Inter', a sans-serif font, for body text to ensure readability and a contemporary look.
- Use simple SVG icons to represent services and contact methods, styled with the gold accent color.
- Employ a grid-based layout for product, portfolio, and service sections to provide a structured and clean appearance, using generous whitespace.
- Implement smooth scrolling and subtle fade/slide-in animations for sections entering the viewport, using IntersectionObserver for a lightweight effect.
